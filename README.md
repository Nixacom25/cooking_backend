Cooking Backend API

Backend REST de l’application Cooked.

Ce serveur gère toute la logique métier de l’application mobile :

 Authentification des utilisateurs
 Gestion des ingrédients (frigo/cuisine)
 Gestion des recettes
 Recommandations personnalisées
 Favoris
Liste de courses (Grocery list)
 Import de recettes

Stack Technique
Backend

Java 17+

Spring Boot 3

Spring Web (API REST)

Spring Data JPA (Hibernate)

Spring Security + JWT

Maven

Lombok

Base de données

PostgreSQL
